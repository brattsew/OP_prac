# Classification

Практическая работа по классификации данных.

В ноутбуке обучаются и сравниваются модели классификации, оценивается качество на тестовой выборке и формируется итоговый файл `output.csv`.

## Состав

- `main.ipynb` — основной ноутбук с анализом, обучением моделей и созданием результата;
- `REPORT.md` — отчёт и ответы на контрольные вопросы;
- `train_data.csv` — обучающая выборка;
- `test_data.csv` — тестовая выборка;
- `public_data.csv` — исходные данные;
- `input.csv` — данные для формирования итогового ответа;
- `output.csv` — результат классификации;
- `pyproject.toml` и `uv.lock` — зависимости проекта.

## Запуск

Установить зависимости:

```bash
uv sync
```

Открыть ноутбук:

```bash
uv run jupyter notebook main.ipynb
```

После выполнения ноутбука создаётся или обновляется файл `output.csv`.

## Результат

В отчёте указано итоговое качество модели и приведены ответы на вопросы по работе.
