# Classification

Практическая работа по классификации данных и построению модели машинного обучения.

Проект оформлен в виде Jupyter Notebook. В ноутбуке выполняется анализ данных, обучение нескольких моделей, сравнение качества и создание итогового файла `output.csv`.

## Что делает программа

Ноутбук:

- загружает данные из CSV-файлов;
- разделяет признаки и целевую переменную;
- обучает модели классификации;
- сравнивает качество моделей на тестовой выборке;
- выбирает лучшую модель;
- дообучает итоговую модель на всех доступных данных;
- формирует файл `output.csv` на основе `input.csv`;
- выводит итоговое качество модели.

## Используемые модели

В работе сравниваются модели из `scikit-learn`:

```text
DecisionTree
RandomForest
```

По результатам выполнения ноутбука лучшей моделью выбрана:

```text
RandomForest
```

Итоговое качество на тестовой выборке:

```text
accuracy = 0.97468
```

## Входные и выходные файлы

Для работы используются файлы:

- `train_data.csv` — обучающая выборка;
- `test_data.csv` — тестовая выборка;
- `public_data.csv` — исходные данные;
- `input.csv` — данные, для которых нужно получить предсказания.

Итоговый файл:

```text
output.csv
```

В него записываются данные из `input.csv` с добавленными предсказаниями модели.

## Установка через uv

Если установлен `uv`, достаточно выполнить:

```bash
uv sync
```

Запуск Jupyter Notebook:

```bash
uv run jupyter notebook main.ipynb
```

## Как пользоваться

Откройте ноутбук:

```text
main.ipynb
```

После запуска ячеек ноутбук последовательно выполнит:

- загрузку данных;
- подготовку признаков;
- обучение моделей;
- оценку качества;
- выбор лучшей модели;
- создание `output.csv`.

После успешного выполнения в консоли ноутбука выводится итоговая модель и значение `accuracy`.
