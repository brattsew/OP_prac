# Time

Практическая работа по прогнозированию временного ряда.

В ноутбуке анализируется временной ряд из `public_data.csv`, подбираются модели прогнозирования и формируется прогноз на 90 дней.

## Состав

- `forecast_timeseries.ipynb` — основной ноутбук с анализом данных, сравнением подходов и прогнозом;
- `public_data.csv` — исходные данные;
- `forecast_90_days.csv` — результат прогноза, создаётся после выполнения ноутбука;
- `pyproject.toml` и `uv.lock` — зависимости проекта.

## Запуск

Установить зависимости:

```bash
uv sync
```

Открыть ноутбук:

```bash
uv run jupyter notebook forecast_timeseries.ipynb
```

После выполнения ноутбука создаётся или обновляется файл `forecast_90_days.csv`.

## Результат

Итоговый файл содержит прогнозные значения на следующие 90 дней.
