# Time

Практическая работа по прогнозированию временных рядов на 90 дней.

Проект оформлен в виде Jupyter Notebook. В ноутбуке анализируются временные ряды из файла `public_data.csv`, сравниваются модели прогнозирования и создаётся итоговый файл `forecast_90_days.csv`.

## Что делает программа

Ноутбук:

- загружает временные ряды из CSV-файла;
- проверяет структуру данных;
- строит графики и анализирует динамику рядов;
- выделяет последние 90 дней как проверочный период;
- сравнивает модели прогнозирования по метрике `MAE`;
- выбирает лучшую модель для каждого временного ряда;
- обучает выбранные модели на всех данных;
- строит прогноз на следующие 90 дней;
- сохраняет результат в файл `forecast_90_days.csv`.

## Используемые модели

В работе сравниваются два подхода:

```text
seasonal_naive_7
ExponentialSmoothing
```

Что они означают:

- `seasonal_naive_7` — повторяет значения за последнюю неделю;
- `ExponentialSmoothing` — модель экспоненциального сглаживания с недельной сезонностью.

Для каждого ряда выбирается модель с меньшей ошибкой `MAE` на последних 90 известных днях.

## Входные и выходные файлы

Исходный файл:

```text
public_data.csv
```

Итоговый файл:

```text
forecast_90_days.csv
```

Итоговая таблица содержит 90 строк прогноза.

## Установка через uv

Если установлен `uv`, достаточно выполнить:

```bash
uv sync
```

Запуск Jupyter Notebook:

```bash
uv run jupyter notebook forecast_timeseries.ipynb
```

## Как пользоваться

Откройте ноутбук:

```text
forecast_timeseries.ipynb
```

После запуска ячеек ноутбук последовательно выполнит:

- загрузку данных;
- анализ временных рядов;
- проверку моделей на последних 90 днях;
- выбор лучшей модели для каждого ряда;
- построение прогноза;
- сохранение `forecast_90_days.csv`.

После успешного выполнения в консоли ноутбука выводится сообщение:

```text
Файл сохранен: forecast_90_days.csv
Количество строк: 90
```
