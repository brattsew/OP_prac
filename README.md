# OP

Репозиторий с тремя практическими работами по обработке данных на Python.

## Работы

| Папка | Описание |
| --- | --- |
| `Classification` | Классификация данных, оценка качества модели и формирование `output.csv`. |
| `Parser` | Парсер русскоязычных новостей из веб-источников с сохранением датасета в XML. |
| `Time` | Прогнозирование временного ряда на основе `public_data.csv`. |

## Запуск

Каждая работа хранится в отдельной папке со своим `pyproject.toml` и `uv.lock`.

Пример запуска через `uv`:

```bash
cd Classification
uv sync
uv run python main.py
```
